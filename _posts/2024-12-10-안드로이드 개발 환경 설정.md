### <span class="larger-text" style="display: block; margin-bottom:-10px;">1. **React-Native**</span>

  <span style="color:rgb(138, 180, 248)">리액트 네이티브 (React Native)</span> 는 모바일 어플리케이션 개발을 위한 오픈 소스 프레임워크입니다.

  자바스크립트와 리액트를 사용하여 iOS와 Android 플랫폼에서 앱을 개발할 수 있게 해줍니다.

  리액트 네이티브를 사용하기 위해선 <span style="color:rgb(138, 180, 248)">React-Native-CLI</span> 를 설치해야 합니다.

  아래의 명령어를 통해 설치를 진행해주세요.

  `npm install -g react-native-cli`

  설치가 완료되었다면 아래의 명령어로 <span style="color:rgb(138, 180, 248)">React-Native-CLI</span> 가 잘 설치되었는지 확인해보세요.

  `npx react-native --version`

---

### <span class="larger-text" style="display: block; margin-bottom:-10px;">2. **Android-Studio 설치**</span>

모바일 어플리케이션을 개발할 때 <span style="color:rgb(138, 180, 248)">안드로이드 스튜디오(Android Studio)</span> 는 안드로이드 에뮬레이터와

안드로이드 빌드 도구 (SDK) 를 제공하기 때문에 필요합니다.

<a href="https://developer.android.com/studio?hl=ko" target="_blank" style="display: inline-block; padding: 10px 20px; border: 2px solid rgb(138, 180, 248); border-radius: 5px; background-color: #f0f0f0; text-decoration: none; color: rgb(118, 160, 228); font-weight: bold; transition: background-color 0.3s, color 0.3s;">
  Android Studio 공식 사이트
</a>

설치가 완료되었다면 <span style="color:rgb(138, 180, 248)">Settings - Languages & Frameworks - Android SDK - SDK Platforms</span>  로 이동

<img src="assets/img/favicons/sdk설치.png" style="border: 2px solid #cccccc; border-radius: 5px;" width="800px"  height="800px" />

Android 14.0 버전 기준
- Android SDK Platforms 34
- Desktop Intel x86_64 Atom System Image
- Intel x86_64 Atom System Image
- Google APIs Intel x86_64 Atom System Image
- Google Play Intel x86_64 Atom System Image
- Google APIs ATD Intel x86_64 Atom System Image

위의 박스에 체크하고 OK를 눌러 설치를 진행합니다.

### <span class="larger-text" style="display: block; margin-bottom:-10px;">3. **환경 변수 설정**</span>

설치가 완료되었다면 SDK 환경변수 설정이 필요합니다.

사용자 변수에는 아래와 같이 입력해줍니다

변수명 - <span style="color:rgb(138, 180, 248)">ANDROID_HOME</span>

변수 값 - <span style="color:rgb(138, 180, 248)">C:\Users\[사용자 이름]\AppData\Local\Android\Sdk</span>

<img src="assets/img/favicons/환경변수설정(1).png" style="border: 2px solid #cccccc; border-radius: 5px;" width="600px"  height="600px" />

시스템 변수의 Path 에는 아래 세 가지를 입력해줍니다.

(1)ㅤ<span style="color:rgb(138, 180, 248)">C:\Users\[사용자 이름]\AppData\Local\Android\Sdk\platform-tools</span>

(2)ㅤ<span style="color:rgb(138, 180, 248)">C:\Users\[사용자 이름]\AppData\Local\Android\Sdk\bin</span>

(3)ㅤ<span style="color:rgb(138, 180, 248)">C:\Users\[사용자 이름]\AppData\Local\Android\Sdk\emulator</span>

<img src="assets/img/favicons/환경변수설정(2).png" style="border: 2px solid #cccccc; border-radius: 5px;" width="600px"  height="600px" />

### <span class="larger-text" style="display: block; margin-bottom:-10px;">4. **가상 머신 생성**</span>

PC 환경에서 모바일 개발 진행 상황을 확인할 수 있는 가상 머신을 생성해보겠습니다.

▼ 안드로이드 스튜디오 우측 상단의 케밥 메뉴를 클릭해 Virtual Device Manager 를 열어 줍니다.

<img src="assets/img/favicons/가상머신(1).png" style="border: 2px solid #cccccc; border-radius: 5px;margin-bottom:10px;" width="800px"  height="800px" />

▼ 좌측 상단의 + 버튼을 눌러 새로운 가상 머신 생성을 진행합니다.

<img src="assets/img/favicons/가상머신(2).png" style="border: 2px solid #cccccc; border-radius: 5px; margin-bottom:10px;" width="800px"  height="800px" />

▼ 가상 머신의 화면 크기를 지정해줍니다. 원하는 크기를 선택하고 Next 를 눌러주세요.

<img src="assets/img/favicons/가상머신(3).png" style="border: 2px solid #cccccc; border-radius: 5px; margin-bottom:10px;" width="800px"  height="800px" />

▼ 원하는 Android 버전을 선택해줍니다. 위에서 Android 14 용 빌드도구를 설치했으므로 14버전을 선택하였습니다.

<img src="assets/img/favicons/가상머신(4).png" style="border: 2px solid #cccccc; border-radius: 5px; margin-bottom:10px;" width="800px"  height="800px" />

▼ 가상 머신의 이름을 설정하고 Finish 버튼을 누르면 생성 완료입니다.

<img src="assets/img/favicons/가상머신(5).png" style="border: 2px solid #cccccc; border-radius: 5px; margin-bottom:10px;" width="800px"  height="800px" />

▼ 재생 버튼을 눌러 가상 머신을 실행시켜줍니다.

<img src="assets/img/favicons/가상머신(6).png" style="border: 2px solid #cccccc; border-radius: 5px;" width="800px"  height="800px" />

### <span class="larger-text" style="display: block; margin-bottom:-10px;">5. **React-Native 프로젝트 생성**</span>

환경 변수 설정이 완료되었다면 아래의 명령어를 통해 React-Native 프로젝트를 생성해주세요.

`npx react-native init 프로젝트이름`

프로젝트가 생성되었다면 안드로이드 스튜디오에서 해당 프로젝트를 열어줍니다.

그리고 해당 프로젝트 루트에서 터미널을 열어 아래의 명령어로 프로젝트를 실행합니다.

`npx react-native run-android`
